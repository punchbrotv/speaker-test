# LinkedIn Summary

I built Speaker Bench, a lightweight browser-based tool for quick speaker checks and calibration.

It generates white noise, pink noise, sine tones, and sine sweeps, with left, right, both, and alternating channel routing. It also includes output level controls, live L/R meters, an optional mic meter for relative room checks, channel pings, and a polarity pulse utility.

Try it here: https://punchbrotv.github.io/speaker-test/
